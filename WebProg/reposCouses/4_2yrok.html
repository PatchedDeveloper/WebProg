Относительное позиционирование элементов на CSS

Относительное позиционирование задается с помощью значения relative для свойства position. Такое позиционирование позволяет сдвигать элементы относительно своего текущего положения на заданную величину. При этом все остальные элементы страницы будут думать, что элемент стоит там, где и стоял изначально. То есть при таком типе позиционирования элемент не выпадает из нормального потока.Смещения от текущего положения задаются свойствами top, right, bottom и left.

Пример 1

Для начала давайте просто сделаем два блока без позиционирования:
  • div id="elem1" /div
  • div id="elem1" /div

    • CSS:
  • #elem1 {
  • width: 100px;
  • height: 100px;
  • border: 1px solid red;
  • }
  • #elem2 {
  • width: 100px;
  • height: 100px;
  • border: 1px solid green;
  • }

    Позиционирование относительно родителя на CSS

    Если элементу задан relative, а его потомку - absolute, то этот потомок будет позиционироваться относительно своего родителя, а не относительно окна браузера.Как правило, в таком случае родителю указывают relative без смещений. В этом случае с этим родителем ничего не происходит, но все его потомки теперь будут позиционироваться относительно него.Давайте посмотрим на примерах.

    Пример 1

    Для начала давайте просто сделаем блок-родитель и блок-потомок без позиционирования:
  • div id="parent
  • div id="child" /div
  • /div

    • CSS:
  • #parent {
  • width: 500px;
  • height: 300px;
  • margin: 0 auto;
  • border: 1px solid red;
  • }
  • #child {
  • width: 200px;
  • height: 200px;
  • border: 1px solid green;
  • }

  • div id="parent
  • div id="child" /div
  • /div
