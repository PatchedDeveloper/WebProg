Установка фонового цвета на CSS

Вы уже умеете менять цвет тексту. Давайте теперь научимся изменять цвет фона. Это делается с помощью свойства background-color, которое принимает те же значения, что и хорошо известное вам color. Давайте зальем блок оранжевым фоном (цвет текста при этом сделаем белым):
  • div id="elem"
  • Lorem ipsum dolor sit amet.
  • /div
    •

    CSS:

  • #elem {
  • width: 300px;
  • height: 100px;
  • color: white;
  • background-color: orange; /* задаем оранжевый цвет фона */
  • }
    •

    Результат выполнения кода:

    Lorem ipsum dolor sit amet.
